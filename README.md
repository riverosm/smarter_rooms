# \<Code Academy\> Nivel 3 - Trabajo final

## Ejecución

```sh
$ git clone "git@github.ibm.com:riverosm/smarter_rooms.git"
$ cd smarter_rooms
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails db:seed
$ rails s
```

Ruby on Rails asignará el puerto configurado y se podrá acceder desde el navegador, por ejemplo:

http://127.0.0.1:3000/
